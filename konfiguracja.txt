═══════════════════════════════════════════════════════════════
SETUP - ROBLOX
═══════════════════════════════════════════════════════════════

1. Otwórz grę w Roblox Studio

2. Włącz HttpService:
   - Game Settings → Security → HttpService ✓ (zaznacz)

3. W ServerScriptService zaktualizuj skrypt "RobloxSync" (lub utwórz nowy):
   - Skopiuj zawartość pliku: roblox-sync.lua
   - Upewnij się, że BACKEND_URL w skrypcie to:
     "https://88602c77-02c7-4b06-8b56-454baca5488c-00-38bejx2g3vlpx.picard.replit.dev"

4. Zapisz zmiany (Publish to Roblox)

═══════════════════════════════════════════════════════════════
TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

❌ "Cannot connect to backend"
   → Czy npm start jest włączony w terminalu?
   → Czy port 3000 jest wolny? (sprawdź: netstat -ano | findstr :3000)

❌ Website się nie zmienia po komendzie
   → Sprawdź F12 → Console czy są błędy
   → Czy HttpService jest włączone w Roblox Studio?
   → Czy RobloxSync script jest w ServerScriptService?

❌ Roblox wysyła "HttpService request failed"
   → Czy HttpService jest włączona w Game Settings?
   → Czy URL w roblox-sync.lua jest prawidłowy?
   → Czy firewall pozwala na komunikację?

❌ Website i Roblox na różnych komputerach
   → Ustaw prawidłowy IP w BACKEND_URL
   → Upewnij się że są w tej samej sieci
   → Sprawdź firewall
